Question:- Scenrio:- a math tools Calculate factorial.
Task:- write a program to compute factorial of a number 
Example:- input:- 5
Output:-120


Answer:-    def factorial(n):
    fact = 1
    for i in range(1, n + 1):
        fact = fact * i
    return fact


num = int(input("Enter number: "))
print(factorial(num))
