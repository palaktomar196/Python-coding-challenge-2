Question:- Scenrio :- A company gives bonuses based on employee salary.
Task:- Write a python program to calculate the bonus amount using gives rules.
Example:- Input:- 45000
                       Output bonus:-- 3150


# Function method


def calculate_bonus(salary):
    bonus = salary * 0.07 # 7% bonus
    return bonus


# Input
salary = float(input("Enter salary: "))


# Function call
bonus = calculate_bonus(salary)


# Output
print("Bonus:", bonus)
