Question:- Scenrio:- A data analytics tools needs to identify the highest value.
Task:- Write a program to find out the largest among three numbers.
Example:- Input:- 10, 25, 15
Output:- 25


Answer:- a = int(input("Enter first number: "))
b = int(input("Enter second number: "))
c = int(input("Enter third number: "))


if a >= b and a >= c:
    print(a)
elif b >= a and b >= c:
    print(b)
else:
    print(c)


Input:-10,25,15
Output:-25
