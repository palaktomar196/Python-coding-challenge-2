Question:- Scenrio:- a system validates palindrome number 
Task:- write a program to check if a number is palindrome 
Example:- input:- 121
Output:- palindrome 


Answer:-   def is_palindrome(n):
    original = n
    rev = 0
    
    while n > 0:
        digit = n % 10
        rev = rev * 10 + digit
        n = n // 10
    
    if original == rev:
        print("Palindrome")
    else:
        print("Not Palindrome")


num = int(input("Enter number: "))
is_palindrome(num)
