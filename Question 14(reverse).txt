Question:- Scenrio:- a security system reverse numeric code 
Task:- write a program to reverse a number.
Example:- input:-1234
Output:-4321


Answer:-  def reverse_number(n):
    rev = 0
    while n > 0:
        digit = n % 10
        rev = rev * 10 + digit
        n = n // 10
    return rev


num = int(input("Enter number: "))
print(reverse_number(num))
